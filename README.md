# Digimon World Dawn & Dusk Companion App
  ## Hosting:
  [![Netlify Status](https://api.netlify.com/api/v1/badges/f3b84f2a-ed61-4104-8ecc-1c48c7356e96/deploy-status)](https://app.netlify.com/projects/digimon-world-dawn-dusk-companion/deploys)
  
  https://digimon-world-dawn-dusk-companion.netlify.app

  ## Features:
  * Digivolution tree for each Digimon in the game.
  * DNA Digivolution combos.
  * "My Team" page that allows the user to mark Digimon as favorites (must have cookies enabled).
  * Animated Digimon via GIFs.

  ## Features to Add:
  * Dawn & Dusk exclusivity marker so the user knows if a Digimon is version exclusive.
  * Digimon lcation hints.

  ## Out of scope:
  * Degeneration evolution data.

  ## Running the code

  Run `npm i` to install the dependencies.
  Run `npm run dev` to start the development server.
